### Instal app

- clone repo
- switch to _dev_ branch
- instal dependenties
  - **npm install / yarn**

### Start dev mode

- **npm run dev / yarn run dev**

### Start production mode

- **npm start / yarn start**

### Build only (_without starting_)

- **npm run build / yarn run build**

### Linting

- **npm run lint / yarn run lint**
